This folder contains a local (offline) copy of **highlight.js** used for client-side syntax highlighting.

Expected files (see `scripts/install-highlightjs.*`):
- `styles/github-dark.min.css`
- `highlight.min.js`
- `languages/csharp.min.js`
- `languages/c.min.js`
- `languages/cpp.min.js`
- `languages/verilog.min.js`

The app will still run if these files are missing, but code highlighting will not work.
